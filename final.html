Features

Light Control

Kira allows you to control light anywhere you want, actually in this version you can only control it from wherever you’re running the node.js server. Remote control or through your phone you can control Kira wherever and whenever you like!

Wifi Enabled

Kira is Wi-Fi enabled, go to the webpage that hosts the Node.js server and you can control Kira just as you would with the remote!

Multi Device Connection

Kira also allows you to have multiple devices connect to the light, making controlling the light accessible for you and whoever needs access!

Concept

House Lighting

This project stems from the frustration I have with the lights in my apartment. I’ve lived here for almost 2 years now, but I still haven’t learned which switch goes to which light. Every time I think I know; I turn on a different light. What also is frustrating is that some light switches go to lights that are across the room, making it a pain to switch on and off lights comfortably. That’s why I wanted to make Kira accessible from anywhere in the room, and from any mobile connected device.

Praticing Web-Development Skills

My career path and passion are web development. I wanted to try and blend my physical prototyping skills with my development skills to see if I could make a usable and attractive product.

Creating Something of Use

Lastly, I wanted to make something I could use. I hate making little websites or knickknacks that get put into GitHub or placed on a shelf.

Arduino Code

Code Implementation

First, we’ll start off with implementing the infrared side of the code. You want to import the IR sensor package which can be found here . Once you have it download add it to your project like I did in the first line of code. Enable the IR sensor in the setup portion, and you’re ready to go.

Now let’s take a look at irBegin() which handles all of the ir sensor data. In this method the inputs are taken care of using a switch statement. The Hex codes that are in the switch statements need to be logged and recorded for your specific remote, that can be done using this guide here. Next let’s look at the web portion of the code. In the web portion we want to be receiving data from the webpage on the state of the light. Here we have an if statement to filter out the null value, -1, and scrub the light brightness.

Lastly look at sending data. In the client-side code we have a method that listens for a read line. So, we want to format our string to be in json so client-side it can be serialized into json without messing up. Then end with a /r/n to signal that the message is finished.

Other than that though, just copy and paste the code and hope that it works.

Client Code

Code Implementation

The client code really is just an exercise in using p5.js. There’s not really much setup here you need to worry about. The main check is to ensure that your com port is the correct one. Here I have it set to COM4 because my Arduino was connected to COM4 but yours might be different. In the Arduino IDE, go to the top menus, click tools, and then port to see which port your Arduino is connected to. Copy and paste that where my COM4 is and you’ll be golden!!
